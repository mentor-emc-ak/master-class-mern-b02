# User Stories

## Authentication

- **US-001** As an employee, I can sign up with my email and password so I can access the app.
- **US-002** As a user, I can log in with my email and password so I can access my boards.
- **US-003** As a user, I can reset my password via email so I can regain access if I forget it.
- **US-004** As a user, I can log out so my session is cleared.

## Boards

- **US-010** As an admin, I can create a new board with a title and description so I can organize a project.
- **US-011** As an admin, I can rename or delete a board I own.
- **US-012** As an admin, I can invite other employees to a board by email.
- **US-013** As an admin, I can set a member's role (member or viewer) on a board.
- **US-014** As an admin, I can remove a user from a board.
- **US-015** As a member/viewer, I can see all boards I have been invited to.

## Lists

- **US-020** As a member, I can create a new list on a board so I can represent a workflow stage.
- **US-021** As a member, I can rename a list.
- **US-022** As a member, I can reorder lists on the board by dragging.
- **US-023** As an admin, I can delete a list (and its cards).

## Cards

- **US-030** As a member, I can create a card in a list with a title.
- **US-031** As a member, I can open a card and add a description, due date, priority, and assignees.
- **US-032** As a member, I can attach labels to a card.
- **US-033** As a member, I can move a card to a different list.
- **US-034** As a member, I can reorder cards within a list.
- **US-035** As a member, I can delete a card I created or am assigned to.
- **US-036** As an admin, I can delete any card on a board I manage.

## Comments

- **US-040** As a member, I can add a comment to a card so I can leave updates.
- **US-041** As a member, I can edit my own comments.
- **US-042** As a member, I can delete my own comments.

## Labels

- **US-050** As an admin or member, I can create a color-coded label on a board.
- **US-051** As a member, I can attach or detach labels from cards.

## My Tasks (Dashboard)

- **US-060** As a logged-in user, I can view all cards assigned to me across all my boards.
- **US-061** As a user, I can filter my tasks by due date, priority, and board.

## Viewer

- **US-070** As a viewer, I can see the board, its lists, and cards but cannot create, edit, or delete anything.
